# apnaastro_landing_page

Public marketing landing page for **ApnaAstro** — the white-label astrology
platform (a DevifAI product). Pitches the platform to astrology businesses who
want their own branded apps.

Single self-contained static page (`index.html`) — no build step, no
dependencies, no external assets. Deploys to Vercel as a static site.

## Deploy (Vercel)
- Framework preset: **Other** (static)
- Build command: *(none)* · Output dir: `.`
- Add your domain, e.g. `apnaastro.devifai.in`.

## Note
This is the PLATFORM's own marketing page. It is different from the per-tenant
app landing pages (those are server-rendered by the backend from each tenant's
brand at `<slug>.<host>`).
