Features

📱

Branded user & astrologer apps

Two Android apps per client, each with their own name, icon, colours and package — built from one codebase.

🎛️

Their own admin panel

Every client gets an admin to manage astrologers, wallets, content, pujas, live sessions and payouts.

🔒

Fully isolated data

Each client runs on its own database. One client can never see another's users, sessions or payments.

🌐

Landing page included

A branded marketing page per client at their own web address — auto-provisioned, no extra work.

Chat, call & live

Their own payments & keys
